# Time Duration Decomposer Web App

A stunning, interactive web application that converts a given number of seconds into a detailed, human-readable duration format (weeks, days, hours, minutes, seconds). Designed with modern aesthetics and smooth animations, this project showcases best practices in frontend development, perfect to impress during job interviews or showcase to the developer community.

---

## Features

- **Beautiful Gradient Design** with a sleek, modern look
- **Responsive Layout** for desktops and mobile devices
- **Animated Result Display** for enhanced user experience
- **User-Friendly Input Validation**
- **Clean, Modular Code** demonstrating advanced JavaScript techniques
- **Instant Conversion** with a single click or pressing Enter
- **Creative Header & Footer** for a complete web experience

## Setup Instructions

### Clone or Download

1. Clone this repository:
```bash
git clone https://github.com/JCaesar45/time-duration-decomposer.git
``
2. Or download the ZIP file and extract it.

### Run Locally

Open `index.html` in your preferred web browser:
```bash
cd path-to-your-cloned-folder
open index.html
``

*(For Windows, double-click `index.html` or open with your browser)*

---

## Usage

1. Enter a positive number of seconds in the input box.
2. Click the **"Convert"** button or press Enter.
3. View the animated, detailed duration output.
